# Userscripts & Tools Collection

这是一个个人自用的效率工具集合，包含浏览器脚本 (Userscripts) 和 PopClip 扩展。

This is a personal collection of efficiency tools, including Userscripts and PopClip extensions.

## 📂 目录 / Contents

| 项目 / Project | 类型 / Type | 描述 / Description |

## 📦 安装 / Installation

### Userscripts (Tampermonkey / Violentmonkey)
1. 进入对应脚本文件夹。
2. 点击 `.user.js` 文件。
3. 点击 "Raw" 按钮，浏览器扩展应会自动识别并提示安装。

### PopClip Extension
1. 进入 `text-editor-popclip` 文件夹。
2. 下载 `.popclipext` 文件（或克隆仓库后双击安装）。
